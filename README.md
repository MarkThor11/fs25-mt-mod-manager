# FS25 MT Mod Manager

A premium desktop application for managing mods, savegames, and system performance for **Farming Simulator 25**.

![App Version](https://img.shields.io/badge/version-1.0.5-brightgreen)
![License](https://img.shields.io/badge/license-MIT-blue)

## 🚀 Key Features

- **Standardized Mod Cards**: A uniform, professional grid layout for all your mods.
- **Savegame Templates**: Create blueprints for your career starts with preset mods and settings.
- **One-Click Optimization**: Automatically tune your `game.xml` for maximum performance based on your hardware.
- **Auto-Update System**: Stay up to date with the latest features automatically.
- **Integrated Radio Stations**: Manage and stream internet radio directly within the manager.
- **Diagnostic Reporting**: Generate technical snapshots for troubleshooting and support.

## 🛠️ Technology Stack

- **Core**: Electron
- **Frontend**: React + Vite
- **Styling**: Vanilla CSS (Premium Dark Mode)
- **Database**: SQLite (via better-sqlite3)
- **Icons**: Lucide React

## 📦 Installation & Build

### Development
```bash
npm install
npm run dev
```

### Build Production
```bash
npm run dist:win
```
